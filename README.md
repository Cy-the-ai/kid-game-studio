## Games

### 1. Counting Carnival 🎪
**Category:** Math | **Ages:** 4-6
- Count falling objects by tapping them
- 3 difficulty levels (Easy: 1-5, Medium: 1-10, Hard: 1-20)
- Text-to-speech counts aloud
- Colorful carnival theme
- Score tracking

### 2. Letter Safari 🦁
**Category:** Reading | **Ages:** 4-6
- Find hidden letters in a jungle scene
- Find 3 target letters among distractors
- Text-to-speech letter pronunciation
- Interactive jungle animals and plants

### 3. Pizza Fractions 🍕
**Category:** Math | **Ages:** 7-9
- Visual fraction learning with pizza slices
- Multiple fractions: ½, ⅓, ¼, ⅙, ⅛
- Click slices to select
- Visual feedback with toppings

### 4. Word Builder 🔤
**Category:** Reading | **Ages:** 4-6
- Spell words by clicking scrambled letters
- Hints provided for each word
- Text-to-speech letter sounds
- Reset and clear options

## Technology Stack

- **HTML5** - Semantic markup
- **JavaScript (Vanilla)** - Game logic and interactivity
- **Bootstrap 5.3** - Responsive design and UI components
- **Font Awesome 6.4** - Icons and visual elements
- **Web Speech API** - Text-to-speech for accessibility

## How to Play

1. Open `index.html` in a web browser
2. Choose a game from the hub
3. Follow the on-screen instructions
4. Click/tap to interact with game elements
